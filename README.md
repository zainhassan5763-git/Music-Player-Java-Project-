# Music Player
A Java-based desktop music player application built using Object-Oriented Programming concepts and Java Swing GUI.

## About

This is an offline music player that allows users to play, pause, stop and navigate through songs. Built as part of my Java OOP course at SSUET (Sir Syed University of Engineering & Technology).

## Features

- Play, Pause and Stop music
- Next and Previous song navigation
- Clean and simple GUI interface
- Supports WAV audio format

##  Technologies Used

- **Java** — Core programming language
- **Java Swing** — GUI framework
- **Java Sound API** — Audio playback
- **OOP Concepts** — Inheritance, Encapsulation, Abstraction, Polymorphism

##  Why Music Files Are Not Included

GitHub has a strict **100MB file size limit** per repository. WAV audio files are extremely large in size — our music folder was **4.23GB** which is way beyond GitHub's limit. Therefore only the source code is uploaded here.

##  How to Add Music Files & Run

**Step 1** — Clone or download this repository
**Step 2** — Open project in **IntelliJ IDEA**
**Step 3** — Create a folder named `Music` inside the project root:
**Step 4** — Add your own `.wav` music files into the `Music/` folder
**Step 5** — Run `GUIMXPLAYER.java`
When downloading make sure to select **WAV format**
